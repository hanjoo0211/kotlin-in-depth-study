# Claude Code 설정

## Kotlin 파일 템플릿

새로운 Kotlin 학습 파일을 만들 때 다음 템플릿을 사용:

```kotlin
package ch{chapter_number}

// {chapter}.{section}.{subsection} 실습

fun main() {
    
}
```

예시:
- `s4_4_1.kt` → `// 4.4.1 실습`
- `s3_2_5.kt` → `// 3.2.5 실습`

파일 위치: `src/main/kotlin/ch{chapter_number}/s{chapter}_{section}_{subsection}.kt`
